Paste chat từ Cursor, Copilot, Claude Code, OpenCode hoặc IDE khác. Tool này tạo file context ngắn gọn để agent tiếp theo hiểu project, quy tắc, việc đang làm và việc cần làm.

Output phù hợp

AGENTS.md, CONTEXT.md, handoff note, README section, JSON archive.

Cách dùng với nhiều IDE

        

          

            
1. Paste chat

            
Lấy các đoạn trao đổi quan trọng từ IDE cũ, issue, terminal log hoặc bug report.

          

          

            
2. Tạo context

            
Tool gom lại thành Markdown có mục tiêu, quy tắc, quyết định, việc tiếp theo.

          

          

            
3. Lưu vào repo

            
Đặt output vào AGENTS.md, CONTEXT.md hoặc docs/handoff.md.
